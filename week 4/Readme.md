# 🏥 AI-Powered Breast Cancer Risk Assessment System

An advanced Machine Learning-based healthcare application that predicts whether a breast tumor is **Benign** or **Malignant** using a Random Forest Classifier and provides an interactive risk assessment dashboard.

## 📌 Project Overview

This project leverages Machine Learning and Streamlit to assist in early breast cancer risk screening. Users can enter clinical measurements and receive a prediction along with risk analysis, performance metrics, visualizations, and a downloadable medical report.

## 🚀 Features

### 🔍 Cancer Prediction

* Predicts whether a tumor is Benign or Malignant
* Probability-based risk assessment

### 📊 Risk Dashboard

* Cancer Risk Percentage
* Benign Probability
* Risk Level Indicator
* Visual Risk Meter

### 📋 Patient Health Report Card

* Prediction Summary
* Risk Level
* Model Performance Metrics
* Date & Time Tracking

### 📈 Data Visualization

* Feature Importance Analysis
* ROC Curve
* Confusion Matrix
* Patient vs Dataset Average Comparison

### 🩺 Early Detection Recommendations

* Personalized recommendations based on risk level
* Low, Moderate, and High-Risk guidance

### 📄 Medical Report Generator

* Downloadable PDF Report
* Includes prediction results and model statistics

## 🛠️ Technologies Used

* Python
* Streamlit

* Pandas
* NumPy

## 📂 Dataset

Wisconsin Breast Cancer Diagnostic Dataset

Target Variable:

* M → Malignant
* B → Benign

## 📊 Model Performance

* Precision: High
* Recall: High
* F1 Score: High
