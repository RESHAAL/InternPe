# 🏏 IPL Winning Team Prediction

A Machine Learning-based web application that predicts the winning team of an IPL match using historical IPL data. The project is built using Python, Scikit-Learn, and Streamlit.

## 📌 Overview

This project analyzes past IPL match data and predicts the likely winner based on match-related factors such as:

* Team 1
* Team 2
* Toss Winner
* Toss Decision
* Match Venue

The prediction is generated using a Random Forest Classifier trained on historical IPL match records.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/IPL-Winning-Team-Prediction.git
```

Move to the project directory:

```bash
cd IPL-Winning-Team-Prediction
```

Install required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run IPL_prediction.py
```
